# Coliseum Skate

Landing page for Coliseum Skate built with Astro and React.

## Setup

```bash
npm install
cp .env.example .env
```

Edit `.env` with your credentials:
- `PUBLIC_WEB3FORMS_ACCESS_KEY` - Get from https://web3forms.com/
- `PUBLIC_STRIPE_PAYMENT_LINK` - Create in Stripe Dashboard
- `PUBLIC_CONTACT_EMAIL` - Your contact email

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Deploy (Render.com)

- Build Command: `npm run build`
- Publish Directory: `dist`
- Add environment variables from `.env.example`
